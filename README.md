# Mint dApp

A multi-step dApp for minting Non-Fungible Tokens (NFTs) directly on the Solana Blockchain. 

## Tech Stack

**Frontend Framework & Styling:**
* [React](https://reactjs.org/) (TypeScript)
* [Vite](https://vitejs.dev/)
* [Tailwind CSS v4](https://tailwindcss.com/)
* [Lucide React](https://lucide.dev/) (Icons)

**Web3 Infrastructure:**
* `@solana/wallet-adapter-react` & `@solana/wallet-adapter-react-ui`
* `@metaplex-foundation/umi` (Core Framework)
* `@metaplex-foundation/mpl-token-metadata` (Program Logic)
* `@metaplex-foundation/umi-uploader-irys` (Decentralized Storage)


### Prerequisites

* Node.js (v18+ recommended)
* Solana Wallet (eg. Phantom)
* Ensure your wallet network is set to **Devnet** and funded with Devnet SOL (you can get free SOL from the [Solana Faucet](https://faucet.solana.com/)).
